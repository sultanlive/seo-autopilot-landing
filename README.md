# SEO Autopilot Landing

Статичный лендинг для концепта автономного SEO-сервиса.

## Файлы
- `index.html` — основная страница

## Как открыть
Открой `index.html` в браузере напрямую.

## Технологии
- HTML
- Tailwind CSS через CDN
- Минимальный JavaScript только для мобильного меню

## Заметки
Если захочешь, можно следующим шагом добавить:
- вторую светлую тему
- анимации
- форму заявки
- отдельную страницу pricing
- англоязычную версию
